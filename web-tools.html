Based on this infographic by Illustrio

I love what Illustro did with compiling all of these web design tools into an infographic. I thought it would be great if that infographic were interactive. This is an angular app that allows you to search their resource list by a category title or the name of a web design tool.

This was quite a bit of information to port into a JSON file, so if you see any spelling mistakes or broken links, call me out.
